## 🌠 What is Chronix?

**Chronix** is a universe-themed, smart **Stopwatch & Timer** web application that blends analog elegance with digital precision — wrapped in a stunning cosmic interface with shooting stars, glassmorphism UI, and smooth animations.

Whether you're tracking your workout ⚡, managing a countdown 🎯, or simply watching time flow — **Chronix** is your companion through the universe of moments.

---

## ✨ Features

### 🕰️ Stopwatch
- 🔵 **Analog Clock Display** — real-time rotating hour, minute & second hands
- 💡 **Digital Readout** — `HH:MM:SS.cs` precision below the clock
- ▶️ **Play / ⏸️ Pause** — smooth icon transition
- 🚩 **Lap Flag** — record & list multiple lap times
- 🔄 **Reset** — returns everything to zero instantly
- 💫 **Pulse ring animation** while the clock is ticking

### ⏱️ Timer
- 🔢 **Custom Input** — set any `HH : MM : SS` duration
- 🌀 **Arc Progress Ring** — visual countdown arc fills in real time
- ▶️ **Play / ⏸️ Pause** with smooth button transitions
- ❌ **Cross Button** — resets the timer back to default
- 🎉 **Done! Flash** — pink highlight when countdown hits zero

### 🌌 Universe Background
- 🌟 **Animated Shooting Stars** — 12 staggered across the sky
- ✨ **Twinkling Star Fields** — multi-layered depth
- 🌫️ **Nebula Glow Blobs** — floating ambient atmosphere
- 🪟 **Glassmorphism Navbar** — frosted blur with active scroll spy

## 🎮 How to Use

### ⏱️ Stopwatch
1. Click the **▶️ Play** button to start the stopwatch
2. The **🚩 Flag** and **🔄 Reset** buttons smoothly appear
3. Click **🚩 Flag** to record a lap time
4. Click **▶️ / ⏸️** to pause and resume
5. Click **🔄 Reset** to clear everything

### ⏰ Timer
1. Enter your desired time in `HH : MM : SS` fields
2. Click **▶️ Play** — the arc ring starts counting down
3. The **❌ Cross** button appears to reset anytime
4. When done, a **🎉 Done!** flash appears
